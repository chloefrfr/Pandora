# Pandora

Pandora is a Minecraft server implementation written in Go

### Prerequisites

Go 1.23 or later

### Installation

1. Clone the repository:

```bash
git clone https://github.com/chloefrfr/Pandora.git
cd pandora
```

2. Build the project:

```bash
go build -o pandora.exe ./cmd/pandora
```

3. Run the server

```bash
./pandora
```
